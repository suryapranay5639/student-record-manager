# Student Record Manager (Fixed Version)

This is a small Python + SQLite project demonstrating CRUD operations.

## How to run

1. Unzip the folder.
2. Open a terminal in the project folder.
3. Run `python main.py`
4. Use the menu to add / view / update / delete students.

Notes:
- `students.db` will be created automatically in the project folder.
- No external packages are required; SQLite is part of the Python standard library.
